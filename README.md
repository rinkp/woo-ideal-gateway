# WooCommerce iDEAL Gateway

Payment gateway for WooCommerce that allows iDEAL via Stripe. This does not require a Chamber of commerce number!

## Current features

* iDEAL payments, quick and hassle free
* Refunds through Stripe
* Supported banks
  * ABN Amro
  * ASN Bank
  * Bunq
  * ING
  * Knab
  * Rabobank
  * Regiobank
  * SNS Bank
  * Triodos Bank
  * Van Lanschot
